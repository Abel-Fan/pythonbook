>## 2 python运算符
>#### 2.1 算术运算符
##### 2.1.1 (+ - * /)
    1、+ *除了加法和乘法以外还可以操作字符串、列表、元组
    2、/ 结果都为浮点型 
##### 2.1.2  (% ** //)
    % 取余
    ** 幂运算
    // 地板除,整数

>#### 2.5 位运算符
```
& 按位与  有0为0，全1为1
| 按位或  有1为1，全0为0
^ 按位异或 不同则为1，相同则为0
>>> 右移   
<< 左移
```
