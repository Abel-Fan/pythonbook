1.4.3 列表型 list

定义

存储一系列相关数据的集合

注意
1、列表可以存储任意数据类型  
2、列表是可变数据   
3、列表可以使用切片  
4、列表可以定义一个元素的列表、空列表、多维列表
列表的访问
list[index]
列表的操作
list.append()  在末尾插入   
list.insert(index,item)  
list.pop(index) 删除  
del list[index] 删除  
list.remove(item) 删除元素  
list.clear() 清空  
list.copy()  浅拷贝  
list.count(item) 计数  
list.extend(list) 扩展  
list.index(item)  返回item下标  
list.reverse() 倒序  
list.sort()  排序
深拷贝浅拷贝
import copy
copy.copy() 浅拷贝  
copy.deepcopy() 深拷贝
推导式
[表达式 for 元素 in 序列 <if 条件>]
生成式
(表达式 for 元素 in 序列 <if 条件>)  
与推导式语法相近只是将 [] 换成 ()
遍历
for item in arr:
    print(item)
for item in arr:
    print(arr.index(item),item)   
for i,item in enumerate(arr):
    print(i,item)

1.4.5 字典型 dict

创建字典
dict1 = {'name':'小白'}     
dict2 = dict(name='小白',sex='男')  
dict3 = dict([('name','小白'),('sex','男')])  
dict4 = dict(zip(['name','sex'],['小白','男']))  
dict5 = dict.fromkeys(['英语','数学','语文'],98)
字典的访问方式
dict[key]
字典常用的内建函数
dict.clear() 清空字典  
dict.copy() 浅拷贝  
dict.fromkeys(keys,value) 快速给key赋相同的值  
dict.get(key[,default]) 获得key的值，如果不存在返回default或None
dict.items()  返回键值序列
dict.keys()   返回键序列
dict.values()  返回值序列
dict.pop(key)  删除key对应的item
del dict[key]
dict.setdefault(key,default)  设置默认值
dict.update() 传入可选参数或其他字典进行更新
遍历
for k in dict1.keys():
    print(k)
for v in dict1.values():
    print(v) 
for k,v in dict1.items():
    print(k,v)
推导式
{k:v for k,v in dict.items() if 条件}
{k for k in dict.keys() if 条件}
字典推导式案例
交换键值
合并大小写
