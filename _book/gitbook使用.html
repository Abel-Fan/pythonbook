gitbook使用

  1. 下载node.js环境
  2. 通过npm（node.js包管理工具，相当于python中的pip）下载gitbook命令
    npm install gitbook-cli -g
  3. 初始化一个gitbook
    gitbook init
  4. 创建一个gitbook
    gitbook build
  5. 生成本地的gitbook
    gitbook serve
